# Football CLI Tracker App

This is a simple command-line football management system built with Python.

## Features
- Manage players (goals, assists, matches)
- Create teams and assign players
- Record matches between teams
- Automatically update player statistics
- View team stats and leaderboards

## Files Used
- `players.json` → stores player data
- `team.json` → stores team data
- `matches.json` → stores match history
- `main.py` → main program file

## How to Run
1. Make sure Python is installed
2. Clone the repository or download the files
3. Run:
   ```bash
   python main.py
